1. Network Connectivity Map (주요 전파 경로)

위험원/위험요인 → 사건 → (연쇄) → 피해 · 좌→우 = cascade 깊이 · 색=타입 · 링크 두께·밝기 = 통과 경로 수(트렁크)

3. Entity Type Heatmap

4. Predicate Distribution

관계(predicate) 코드별 빈도

5. Predicate Heatmap

6. Sankey Diagram

축 흐름: 위험원 축 → 관계 → 피해 축

7. Force-directed Graph

전체 그래프 · 색=타입 · 크기=연결도 · 드래그/줌

8. Centrality Ranking
